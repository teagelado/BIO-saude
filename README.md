Uma Landing Page moderna, responsiva e interativa para uma clínica médica fictícia, focada em experiência do usuário (UX) e agendamento simplificado.

---

## 🚀 Sobre o Projeto

O **BioSaúde** foi desenvolvido para praticar conceitos avançados de Front-end, incluindo layouts dinâmicos, manipulação de DOM com JavaScript e design responsivo (Mobile First).

### 🛠 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização avançada, Grid Layout, Flexbox e animações de Hover.
* **JavaScript (ES6+):** Lógica de agendamento, Scroll Reveal e efeitos dinâmicos.
* **Google Fonts:** Tipografia limpa com *Segoe UI*.

---

## ✨ Funcionalidades

* **Design Responsivo:** Adaptável para Celulares, Tablets e Desktops.
* **Cards Interativos:** Ao passar o mouse nos serviços, um card de detalhes com degradê é revelado via CSS Transition.
* **Efeito de Digitação:** O título principal no "Hero" possui uma animação de escrita automática.
* **Scroll Reveal:** Os elementos surgem suavemente conforme o usuário navega pela página.
* **Formulário de Agendamento:** Validação de campos e feedback visual de sucesso sem recarregar a página.
* **Footer Estruturado:** Informações de contato e localização organizadas.

## 🔧 Como Executar o Projeto

1.  Faça o clone do repositório:
    ```bash
    git clone [https://github.com/seu-usuario/biosaude-clinica.git](https://github.com/seu-usuario/biosaude-clinica.git)
    ```
2.  Navegue até a pasta do projeto:

3.  Abra o arquivo `index.html` no seu navegador de preferência.
